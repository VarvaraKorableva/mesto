# Проект: Место
# Single Page Application позволяет делиться своими фотографиями, менять аватар и имя пользователя, ставить и удалять лайки фотографиям, а так же открывать их в увеличенном виде.

### Используемые технологии

* FlexBox
* Grid
* DOM
* JavaScript(ООП)
* Webpack
* Babel
